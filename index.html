Apa itu Next.js?


Next.js adalah framework open-source untuk membuat aplikasi web menggunakan bahasa pemrograman JavaScript. Sederhananya, Next.js memudahkan pengembangan aplikasi web dengan menyediakan sejumlah fitur dan fungsi yang dapat mempercepat proses pembuatan dan pengembangan.
Bagaimana sejarah Next.js?


Next.js pertama kali dirilis di GitHub pada 25 Oktober 2016. Dikembangkan berdasarkan enam prinsip, termasuk fungsionalitas siap pakai tanpa konfigurasi tambahan, penggunaan JavaScript di semua aspek, dan pembagian kode otomatis. Versi 2.0, pada Maret 2017, membawa perbaikan untuk situs web kecil dan meningkatkan efisiensi pembangunan. Versi 7.0, pada September 2018, meningkatkan penanganan kesalahan dan mendukung React's context API untuk penanganan rute dinamis. Versi ini juga upgrade ke Webpack 4. Versi 8.0, pada Februari 2019, menjadi yang pertama menawarkan implementasi serverless, memecah kode menjadi fungsi lambda dan mengurangi waktu serta sumber daya untuk ekspor statis. Pada Maret 2020, versi 9.3 hadir dengan optimisasi dan dukungan untuk Sass global dan modul CSS. Versi 9.5, pada Juli 2020, menambahkan regenerasi statis bertahap, rewrites, dan dukungan pengalihan. Versi 11, pada Juni 2021, membawa dukungan Webpack 5, "Next.js Live" untuk kolaborasi waktu nyata, dan fungsi eksperimental konversi dari Create React App ke Next.js. Next.js 12, pada Oktober 2021, menambahkan compiler Rust, dukungan AVIF, Edge Functions & Middleware, dan Native ESM & URL Imports. Pada 26 Oktober 2022, versi 13 membawa pola routing baru, App Router, serta dukungan untuk React Server Components, streaming, dan metode pengambilan data baru. Pada Mei 2023, versi 13.4 stabil menghadirkan App Router. Oktober 2023 melihat rilis Next.js 14 dengan peningkatan manajemen memori menggunakan runtime edge. Pada Desember 2023, versi 14.0.4 menambah fitur seperti pelacakan permintaan server dan penggunaan memori.
